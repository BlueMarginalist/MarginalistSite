An interactive tool for exploring U.S. Treasury yield curve dynamics from
1990 to present.

## Features

- **Historical playback** — scrub through time to watch the curve shift during
  rate cycles, crises, and QE programs.
- **Spread calculator** — click any two tenors to track the spread over time
  (2s10s, 3m10y, etc.) with automatic inversion highlighting.
- **Recession overlays** — NBER recession bands drawn automatically for
  contextual reference.
- **Export** — download any chart as SVG or copy the underlying data as CSV.

## How it works

Data is pulled from the [FRED API](https://fred.stlouisfed.org/) at build time
and cached as static JSON. The frontend is vanilla TypeScript + D3.js with no
framework overhead — the entire bundle is under 80 KB gzipped.

The embedded demo above is hosted at `tools.jackyyang.com`. If you prefer to
run it locally:

```bash
git clone https://github.com/jackyyang/yield-curve-explorer
cd yield-curve-explorer
npm install
npm run dev
```

## Technical notes

Yield data uses constant-maturity Treasury (CMT) rates from FRED series
`DGS1MO` through `DGS30`. Missing values (holidays, weekends) are
forward-filled before rendering.
